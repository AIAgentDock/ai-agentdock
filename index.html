Cursor Rules

Persistent instructions for Cursor Agent and Chat. Place them in .cursor/rules/ as .mdc or Markdown files at your repo root.

Cursor applies them automatically. Open any rule page and click Copy Rule to paste into your project.

See Cursor Agent & Skills rules for workflow tips.

Windsurf Rules

Persistent instructions for Windsurf Cascade sessions. Open Windsurf → Rules panel → import or paste the rule text.

Windsurf uses these across all Cascade conversations. Browse the directory, copy a rule, and paste into the Rules panel.

Tip: Trim sections after copying to fit your project and avoid overrunning the AI context window.

FAQ

Where do Cursor rules go?

Put project rules in .cursor/rules/ at your repo root. Use .mdc files (recommended) or Markdown. Cursor applies them automatically in Agent and Chat. See our Cursor Agent & Skills rules for workflow tips.

Where do Windsurf rules go?

Open Windsurf → Rules panel → import or paste the rule text. Windsurf uses these as persistent instructions across Cascade sessions.

How do I use a rule from AI Agent Dock?

Find a rule in the directory, open its detail page, click Copy Rule, then paste into your editor. Each rule has a permanent URL you can bookmark or share (e.g. /rules/fastapi-backend).

Can I edit rules after copying?

Yes. Rules are starting templates — remove sections you do not need, add project-specific conventions, and keep total length reasonable so you do not overrun the AI context window.

Are these rules free to use?

Yes. Most rules on this site are MIT-licensed unless noted otherwise on the rule page. Check the License line on each card or detail page.

How do I submit my own rule?

Visit Submit a Rule for guidelines, then send your rule by email.
