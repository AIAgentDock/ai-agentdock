How to use Cursor Rules

Persistent instructions for Cursor Agent and Chat. Place project rules in .cursor/rules/ as .mdc or Markdown files at your repo root — for example .cursor/rules/*.mdc. Cursor applies them automatically in Agent and Chat.

Open any rule page and click Copy Rule to paste into your project. See Cursor Agent & Skills rules for workflow tips.

About Cursor Rules

What are Cursor rules?

Cursor rules are persistent instructions that shape how Cursor Agent and Chat write, review, and refactor code in your project. They encode team conventions, stack-specific patterns, and guardrails so the AI stays consistent session to session.

Where should Cursor rules be stored?

Save project rules as .mdc or Markdown files in .cursor/rules/ at your repo root — for example .cursor/rules/*.mdc. Cursor applies them automatically in Agent and Chat.

Best Cursor rules by stack

  • Fullstack / frontend: Next.js Full-Stack, React Frontend, Tailwind UI, shadcn/ui
  • Backend / data: Supabase Database, Prisma ORM
  • AI / agents: Vercel AI SDK, MCP Server, Cursor Agent & Skills
  • DevOps / quality: Docker, GitHub Actions CI, Playwright E2E

FAQ

Where do Cursor rules go?

Put project rules in .cursor/rules/ at your repo root. Use .mdc files (recommended) or Markdown — for example .cursor/rules/*.mdc. Cursor applies them automatically in Agent and Chat. See our Cursor Agent & Skills rules for workflow tips.

How do I use a rule from AI Agent Dock?

Browse the Cursor directory, open a rule detail page, click Copy Rule, then paste into .cursor/rules/. Each rule has a permanent URL (e.g. /rules/nextjs-fullstack).

Can I edit rules after copying?

Yes. Rules are starting templates — remove sections you do not need, add project-specific conventions, and keep total length reasonable so you do not overrun the AI context window.

Are these rules free to use?

Yes. Most rules on this site are MIT-licensed unless noted otherwise on the rule page. Check the License line on each card or detail page.

How do I submit my own rule?

Visit Submit a Rule for guidelines, then send your rule by email.

Looking for Windsurf / Devin Desktop rules instead?

Windsurf rules go in .devin/rules/*.md (preferred) or legacy .windsurf/rules/*.md. Browse the Windsurf directory or see Also available: Windsurf Rules above.
