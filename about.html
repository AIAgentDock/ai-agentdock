About AI Agent Dock

A free, curated directory of copy-ready AI coding assets for modern development workflows.

What Is AI Agent Dock?

AI Agent Dock is an open directory of copy-ready AI coding assets — including rules, roles, skills, hooks, workflows and MCP configs — for AI-powered code editors and agents such as Cursor, Windsurf, Claude Code, GitHub Copilot, Codex and MCP-compatible workflows. Instead of writing prompts from scratch, you can browse, search, and copy templates tailored to popular stacks — Next.js, Python, Supabase, Tailwind CSS, Rust, and more.

Each asset type serves a different purpose: Rules are persistent project instructions; roles set the AI's persona; skills guide multi-step tasks; hooks are short pre-task prompts; workflows walk through complex processes; and MCP configs connect agents to external tools.

Supported Tools

  • Cursor — rules in .cursor/rules/, plus roles, skills and hooks for Agent & Chat
  • Windsurf — rules in .devin/rules/, plus Cascade skills and workflows
  • Claude Code — project instructions, roles, skills and hooks
  • GitHub Copilot — copilot-instructions.md prompts and Copilot Chat workflows
  • Codex — system prompts, roles, skills and deployment workflows
  • MCP — server configs and integration workflows for Model Context Protocol

Why Use It?

  • Save time — start from battle-tested asset templates instead of a blank prompt
  • Stay consistent — align AI output with your stack and coding style
  • One-click copy — grab an asset and paste it into your editor in seconds
  • Filter by tool and type — find rules, roles, skills, hooks, workflows or MCP configs for your agent

Who Is It For?

AI Agent Dock is built for individual developers, freelancers, and small teams who rely on AI-assisted coding and want better, more predictable results. Whether you are bootstrapping a side project or standardizing conventions across a codebase, curated assets help your AI understand what "good code" means in your context.
